Abrir `tasacion.html` directamente en el navegador, o publicar en GitHub Pages:

1. Ir a **Settings → Pages**
2. Source: `Deploy from a branch` → `main` → `/ (root)`
3. La página quedará disponible en `https://usuario.github.io/repositorio/tasacion.html`
